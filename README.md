# ARomaMenu

## Steps
(1) Clone the repo
(2) Navigate the Backend dir
(3) install node modules 
```
npm install
```
(4) Make changes in env file
(5) Run the backend server
```
npm run dev
```
(6) Navigate to frontend dir
(7) Unzip the android folder and node_modules
(8) Start the frontend
```
npm start
```
